Beautiful Syntax

Inspired by Lua and Ruby, Tourmaline features clean, readable syntax that's perfect for beginners and enjoyable for experts.

Easy to Learn

Start coding in minutes! Tourmaline's intuitive design makes programming accessible to everyone, regardless of experience level.

Powerful Features

Despite its simplicity, Tourmaline includes lists, dictionaries, functions, loops, and all the tools you need for real projects.

Getting Started

  1. Clone the repository from GitHub
  2. Save your code in a .trm file
  3. Run with: python tourmaline.py yourfile.trm or after adding tourma.bat to PATH, tourma yourfile.trm

Code Examples

Hello World & Variables

Start with the basics - variables, strings, and output.

# Your first Tourmaline program
let name = "World"
print("Hello, " + name + "!")

let x = 42
let y = 3.14
let is_awesome = true

Functions

Create reusable code with functions.

function greet(person)
    print("Hello, " + person + "!")
end

function add(a, b)
    let result = a + b
    print("Sum: " + str(result))
end

greet("Alice")
add(10, 20)

Lists & Loops

Work with collections and iterate through them.

# Lists are easy!
let fruits = ["apple", "banana", "cherry"]

for fruit in fruits
    print(fruit)
end

# While loops
let counter = 0
while counter < 5
    print("Count: " + str(counter))
    counter += 1
end

Conditionals

Make decisions with if/elif/else statements.

let score = 85

if score >= 90
    print("Grade: A")
elif score >= 80
    print("Grade: B")
elif score >= 70
    print("Grade: C")
else
    print("Grade: F")
end

Dictionaries

Store and access key-value pairs.

# Create a dictionary
let person = {"name": "Bob", "age": 30, "city": "NYC"}

print("Name: " + person["name"])
print("Age: " + str(person["age"]))
print("City: " + person["city"])

Math Functions

Built-in mathematical operations for your calculations.

# Math is built right in!
print(sqrt(16))        # 4.0
print(pow(2, 3))      # 8
print(abs(-5))        # 5
print(min(3, 7, 2))  # 2
print(max(3, 7, 2))  # 7
print(floor(3.7))     # 3
print(ceil(3.2))      # 4
